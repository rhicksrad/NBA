About this snapshot

The data is generated from TeamHistories.csv in this repository using scripts/build_snapshot.mjs. The script keeps the dataset lightweight for the browser and powers the live filters and charts on this page.

Looking ahead, this playground can ingest player and game level datasets to power richer visuals—this snapshot is the first step toward that MVP.
