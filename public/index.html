About this hub

The NBA Intelligence Hub combines franchise, schedule, player, game, team, and leaderboard snapshots built from the CSV sources in this repository.

Use the scripts in the scripts/ directory (build_snapshot.mjs, build_schedule_snapshot.mjs, and build_insights.py) to regenerate the snapshots and refresh every panel with the latest data.
