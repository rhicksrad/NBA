Roadmap to launch

This experience hub is designed to scale. Each milestone below unlocks new sections, data models, and visualization frameworks.

  • Phase 1 — Data plumbing Integrate official league feeds, clean historical tables, and normalize player IDs.
  • Phase 2 — Visual infrastructure Ship reusable chart components with performance budgets for rich interactivity.
  • Phase 3 — Storytelling Release narrative walkthroughs that combine metrics with editorial context.
  • Phase 4 — Personalization Enable saved filters, watchlists, and automated scouting briefs for each franchise.
