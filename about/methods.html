Methods & provenance

NBA Pulse publishes compact visual summaries backed by static JSON snapshots. Each module lists its source badge and update cadence on the main dashboard. This page captures the provenance and caveats for every feed.

Data lineage

Module Source Refresh cadence Notes
Offseason shifts NBA News Desk Updated daily Index scores blend roster moves, coaching updates, and health designations.
Milestones sparkline NBA Advanced Stats Updated daily at 09:00 UTC Trend uses the last ten completed games and smooths out back-to-backs.
Countdown checkpoints NBA Schedule Service Updated weekly on Mondays Workload ratings reflect scrimmage volume and travel between preseason hubs.

Caveats & responsible use

  • Snapshots are stored in this repository under data/ for auditing and reproducibility.
  • No personally identifiable information is collected; all player data is sourced from public league feeds.
  • Historical comparisons normalize scoring environments to maintain season-over-season comparability.
  • Users should verify context with official NBA communications before making commercial decisions.

Refresh workflow

Static JSON files are regenerated locally via pnpm run build, which compiles the visualization helpers and copies the latest data snapshots. The GitHub Actions Pages workflow validates links, schemas, and Lighthouse budgets before deployment.
